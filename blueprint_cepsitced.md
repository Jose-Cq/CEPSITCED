# Blueprint de Estrategia y Arquitectura Técnica: Landing Page CEPSITCED

---

## 1. Resumen Ejecutivo
Este documento presenta el blueprint estratégico y técnico para la implementación de la landing page del **Centro Psicológico Terapéutico Clínico Educativo (CEPSITCED)**. El diseño de este sitio de una sola página (landing page) se centra en maximizar la conversión de usuarios (padres de familia y adultos) en pacientes mediante una experiencia accesible, empática y de alto rendimiento. Construido exclusivamente en HTML5, CSS3 y JavaScript vanilla (sin frameworks ni dependencias pesadas), el blueprint balancea los requisitos éticos de la salud mental con técnicas modernas de marketing de respuesta directa (fórmula PAS). Se garantiza la optimización para Core Web Vitals, accesibilidad WCAG AA y SEO local estratégico, proporcionando una estructura lista para codificación.

---

## 2. Público Objetivo
La comunicación y segmentación de la landing page se dividirá en dos perfiles de usuarios claramente definidos:

### Perfil A: Padres de familia (Foco: Área Educativa, Infantil y Lenguaje)
*   **Dolores Emocionales y Educativos:** Sentimiento de culpa ("¿habré hecho algo mal?"), incertidumbre ante el futuro de su hijo, frustración por bajo rendimiento escolar o problemas de comunicación, miedo al estigma social del psicólogo.
*   **Intenciones de Búsqueda (Keywords):** 
    *   *“terapia de lenguaje niños cerca de mí”*
    *   *“problemas de aprendizaje psicólogo infantil”*
    *   *“cómo saber si mi hijo necesita terapia de lenguaje”*
    *   *“dificultades de atención y concentración colegios”*
*   **Objeciones Comunes:** 
    *   *“Es muy costoso a largo plazo.”* (Mitigación: Mostrar facilidades, enfoque de evaluación de triaje y planes integrales).
    *   *“Mi hijo se sentirá etiquetado o enfermo.”* (Mitigación: Enfoque lúdico, empático y centrado en el desarrollo de habilidades, no en la patologización).
    *   *“No tengo tiempo por el trabajo o la escuela.”* (Mitigación: Flexibilidad horaria por las tardes y sábados).

### Perfil B: Adultos y Jóvenes (Foco: Área Clínico e Individual)
*   **Dolores Emocionales y Clínicos:** Agobio crónico por ansiedad, tristeza persistente (depresión), problemas de relación o de pareja, sensación de estancamiento académico/profesional, dificultad para regular sus emociones.
*   **Intenciones de Búsqueda (Keywords):** 
    *   *“terapia para ansiedad y estrés precio”*
    *   *“psicólogo clínico para adultos lima”*
    *   *“cómo superar una crisis de ansiedad”*
    *   *“consultas psicológicas confidenciales online”*
*   **Objeciones Comunes:** 
    *   *“La terapia no me va a funcionar.”* (Mitigación: Explicar el enfoque científico/clínico y mostrar casos de superación de manera de forma ética y anónima).
    *   *“Me da vergüenza contar mis problemas a un extraño.”* (Mitigación: Énfasis absoluto en la confidencialidad garantizada y ambiente seguro).
    *   *“No sé si puedo pagarlo constantemente.”* (Mitigación: Transparencia en el proceso y establecimiento de objetivos claros desde el inicio para evitar terapias eternas).

## 4. Oferta y Call To Action (CTA)
*   **Oferta de Entrada (Lead Magnet Ético):**
    *   **"Primer contacto de orientación y triaje telefónico de 15 minutos sin costo"**. 
    *   *Justificación ética:* No se realiza un diagnóstico ni se prescribe tratamiento durante esta llamada. Actúa como un filtro clínico donde un profesional escucha el caso brevemente para derivarlo con el terapeuta de la especialidad correcta, reduciendo el temor inicial del paciente de equivocarse de especialista o gastar en vano.
*   **CTA Principal:** 
    *   Texto: **"Coordinar mi orientación de 15 minutos sin costo"**
    *   Acción: Abre chat de WhatsApp con un mensaje predefinido ("*Hola CEPSITCED, me gustaría coordinar la llamada de orientación sin costo. Mi caso es sobre...*").
*   **CTA Secundario:** 
    *   Texto: **"Solicitar llamada de pre-admisión"** (Formulario web para contacto telefónico o por correo).
*   **Razonamiento Psicológico del Diseño:**
    *   **Azul Clínico Sereno:** Transmite profesionalismo, estabilidad, rigor médico y confidencialidad. Reduce el estado de alerta cerebral.
    *   **Celeste Calma:** Aporta ligereza visual, transmite accesibilidad emocional y luz. Evita que la página luzca sombría o demasiado fría.
    *   **Verde WhatsApp:** Destaca visualmente sobre la paleta de azules, representando accesibilidad, inmediatez y una zona de comunicación familiar y segura (baja fricción).

## 5. Arquitectura de la Landing Page (Estructura Seccional)

### Sección 1: Hero (Primer pliegue)
*   **Estructura Visual:** Distribución a dos columnas en escritorio (una columna en móviles).
*   **Elementos:**
    *   H1 llamativo, empático y orientado a la solución.
    *   Subcopy que segmenta y tranquiliza.
    *   3 checks con iconos de confianza (ej. *"Profesionales Colegiados"*, *"Sesiones Presenciales y Online"*, *"Confidencialidad Absoluta"*).
    *   Botón CTA principal (WhatsApp) destacado con micro-interacción suave.
    *   Sello de confianza visual: Icono de Colegio de Psicólogos (o insignia de habilitación profesional).
    *   Fotografía de fondo o ilustración real del centro transmitiendo calidez.

### Sección 2: Áreas de Intervención / Servicios (PAS - Fase de Solución)
*   **Estructura Visual:** Grid de 3 tarjetas interactivas (hover con elevación sutil).
*   **Contenido:**
    *   **Tarjeta 1: Área Clínica (Adultos y Parejas):** Tratamiento de ansiedad, depresión, manejo de estrés, desarrollo personal y terapia de pareja.
    *   **Tarjeta 2: Área Educativa (Niños y Adolescentes):** Diagnóstico de problemas de aprendizaje, orientación vocacional, técnicas de estudio y hábitos conductuales.
    *   **Tarjeta 3: Área Terapéutica de Desarrollo:** Terapia de lenguaje, psicomotricidad y terapia ocupacional infantil.
*   **Cada tarjeta incluye:** Icono personalizado SVG, breve descripción de síntomas comunes y enlace sutil ("Saber más de esta área" que desplaza al usuario al formulario de contacto con esa opción seleccionada).

### Sección 3: Nuestro Enfoque / Cómo Trabajamos
*   **Estructura Visual:** Línea de tiempo (timeline) vertical en móvil y horizontal en escritorio.
*   **Pasos:**
    1.  **Contacto Directo:** Nos escribes por WhatsApp o completas el formulario y un especialista te atiende para tu llamada de orientación inicial.
    2.  **Evaluación Personalizada:** Agendamos una sesión diagnóstica para comprender tu situación particular y metas terapéuticas.
    3.  **Plan de Intervención:** Diseñamos un camino estructurado con sesiones periódicas y herramientas prácticas para el día a día.
    4.  **Seguimiento y Alta:** Monitoreamos tu evolución de manera continua hasta que alcances las herramientas para avanzar de manera independiente.

### Sección 4: Impacto y Metodología (Prueba Social Ética)
*   *Nota Ética:* De acuerdo al Código de Ética del Colegio de Psicólogos, está prohibido publicar testimonios comerciales que expongan la identidad del paciente o que realicen promesas exageradas de curación.
*   **Enfoque de la Sección:** Presentar casos de estudio redactados de forma clínica, anónima e ilustrativa (ej. *"Caso A: Superación del bloqueo por ansiedad ante exámenes en adolescentes"* o *"Caso B: Integración social y mejoría en el habla en terapia de lenguaje infantil"*).
*   **Estructura:** Bloques deslizantes tipo carrusel o pestañas interactivas limpias que explican: El problema detectado, la intervención realizada y el resultado cualitativo/bienestar obtenido.

### Sección 5: El Centro y Equipo
*   **Estructura Visual:** Sección de dos columnas.
    *   Columna Izquierda: Foto del espacio del consultorio (iluminación natural, colores neutros, espacio ordenado y acogedor para niños).
    *   Columna Derecha: Tarjetas del staff principal con nombres placeholders (ej. *"Ps. [Nombre] [Apellido]"*), su especialidad, código CBP y RNE.
*   **Texto de Confianza:** Breve reseña del compromiso del equipo con la salud mental comunitaria y el trato humanizado.

### Sección 6: Gestión de Citas y Transparencia
*   **Estructura Visual:** Panel dividido de lectura rápida.
    *   **Horarios:** Tabla limpia con horas de atención presencial y virtual (Lunes a Sábado).
    *   **Flexibilidad:** Explicación de la política de reprogramación (gratuita hasta 24 horas antes) y puntualidad.
    *   **Transparencia de Costos:** Mensaje claro de que los presupuestos de evaluación se coordinan previamente, sin cobros sorpresa o tarifas ocultas.

### Sección 7: FAQ (Preguntas Frecuentes)
*   **Estructura Visual:** Acordeón colapsable vertical implementado con etiquetas HTML `<details>` y `<summary>` (altamente accesibles) o JS dinámico.
*   **Contenido:** 10 preguntas frecuentes que neutralizan las objeciones más comunes (se detallan copys en sección 6).

### Sección 8: Contacto (Conversión Directa)
*   **Estructura Visual:** Dos columnas.
    *   Columna A: Formulario de pre-admisión (Nombre, Teléfono de contacto, Tipo de consulta [Dropdown], Mensaje corto).
    *   Columna B: Datos de contacto directo (Teléfono, Correo institucional, Dirección física con mapa interactivo o placeholder de mapa en imagen liviana, e icono destacado de WhatsApp).

### Sección 9: Footer (Legales y Emergencias)
*   **Estructura Visual:** Fondo oscuro sereno (Gris Oxford) con tres secciones de enlaces:
    1.  **Información Legal:** Política de privacidad, tratamiento de datos personales (Ley N° 29733 de Protección de Datos Personales), términos de servicio de teleconsulta.
    2.  **Advertencia de Crisis (Crítico):** *"IMPORTANTE: Si te encuentras en una situación de emergencia extrema, peligro de autolesión o crisis psiquiátrica inmediata, comunícate gratuitamente con la línea nacional [Línea 113 opción 5 en Perú / Teléfono de emergencia local] o acude al centro de salud más cercano. No somos un servicio de urgencias médicas."*

## 7. UI/UX: Guía Visual y Razón Psicológica del Diseño

```
Paleta de Colores
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│ Azul Clínico    │ │ Celeste Calma   │ │ Fondo Suave     │ │ Verde Convers.  │
│ #1E3A8A         │ │ #0EA5E9         │ │ #F8FAFC         │ │ #22C55E         │
│ (Seguridad)     │ │ (Esperanza)     │ │ (Claridad/Respi)│ │ (WhatsApp)      │
└─────────────────┘ └─────────────────┘ └─────────────────┘ └─────────────────┘
```

*   **Paleta de Colores (Basada en HSL para Flexibilidad):**
    *   **Color Primario (Dominante 60%):** `#1E3A8A` / `hsl(224, 64%, 33%)` (Azul Clínico). Genera autoridad, calma el pulso y simboliza rigor científico. Usado en encabezados principales (H1, H2), fondo del footer y bordes estructurales.
    *   **Color Secundario (30%):** `#0EA5E9` / `hsl(198, 93%, 47%)` (Celeste Calma). Aporta ligereza visual, transmite accesibilidad emocional y luz. Usado en subtítulos, iconos informativos, y estados hover de elementos no críticos.
    *   **Color de Fondo:** `#F8FAFC` / `hsl(210, 40%, 98%)` (Gris/Azul muy claro). Evita la fatiga ocular del blanco puro y otorga un aire premium y limpio (clínico pero acogedor).
    *   **Color de Acento (Acciones):** `#22C55E` / `hsl(142, 72%, 45%)` (Verde Conversión). Reservado exclusivamente para el botón flotante de WhatsApp y CTAs de agendamiento instantáneo. Contraste óptimo y familiaridad de uso.
    *   **Color del Texto de Lectura:** `#1E293B` / `hsl(215, 28%, 17%)` (Gris Carbón). Garantiza una legibilidad perfecta cumpliendo con el ratio de contraste WCAG AA de al menos 4.5:1.
*   **Tipografía Sugerida:**
    *   Familia de Fuentes: `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial, sans-serif`.
    *   *Razón Psicológica y Técnica:* Al no cargar fuentes externas de Google Fonts, la página se visualiza de forma instantánea sin parpadeos de fuente (FOIT) y es perfectamente legible por lectores de pantalla y usuarios con dislexia.
*   **Espaciado y Layout (Aire Visual):**
    *   Uso de paddings generosos (`section { padding: 5rem 1rem; }` en escritorio, `3rem 1rem` en móvil). 
    *   El "aire" o espacio en blanco reduce la sobrecarga cognitiva del usuario, quien frecuentemente llega al sitio en un estado de vulnerabilidad emocional o estrés por problemas familiares.

#### A. Navegación Suave (Smooth Scroll) con Ajuste de Altura de Cabecera
```javascript
// main.js - Smooth scroll con offset de cabecera fija
document.querySelectorAll('a[href^="#"]').forEach(anchor => {
    anchor.addEventListener('click', function (e) {
        e.preventDefault();
        const targetId = this.getAttribute('href');
        const targetElement = document.querySelector(targetId);
        
        if (targetElement) {
            const headerOffset = document.querySelector('.main-header').offsetHeight || 70;
            const elementPosition = targetElement.getBoundingClientRect().top;
            const offsetPosition = elementPosition + window.pageYOffset - headerOffset;
            
            window.scrollTo({
                top: offsetPosition,
                behavior: 'smooth'
            });
            
            // Foco accesible al elemento de destino
            targetElement.setAttribute('tabindex', '-1');
            targetElement.focus({ preventScroll: true });
        }
    });
});
```

#### B. Botón de WhatsApp Sticky (CTA Persistente con Sensor de Scroll)
```javascript
// main.js - Mostrar/ocultar CTA móvil según el scroll (evitar tapar el primer pliegue)
const stickyCta = document.querySelector('.sticky-whatsapp-cta');

if (stickyCta) {
    window.addEventListener('scroll', () => {
        // Se muestra únicamente después de hacer scroll de 400px hacia abajo
        if (window.scrollY > 400) {
            stickyCta.classList.add('visible');
            stickyCta.setAttribute('aria-hidden', 'false');
        } else {
            stickyCta.classList.remove('visible');
            stickyCta.setAttribute('aria-hidden', 'true');
        }
    });
}
```

#### C. Acordeón FAQ Accesible
```javascript
// main.js - Control interactivo y accesible del acordeón de FAQ
const faqHeaders = document.querySelectorAll('.faq-header');

faqHeaders.forEach(header => {
    header.addEventListener('click', () => {
        const item = header.parentElement;
        const body = item.querySelector('.faq-body');
        const isExpanded = header.getAttribute('aria-expanded') === 'true';
        
        // Colapsar otros activos (opcional, comportamiento de acordeón tradicional)
        document.querySelectorAll('.faq-item').forEach(otherItem => {
            if (otherItem !== item) {
                otherItem.classList.remove('active');
                otherItem.querySelector('.faq-header').setAttribute('aria-expanded', 'false');
                otherItem.querySelector('.faq-body').style.maxHeight = null;
            }
        });
        
        // Toggle actual
        item.classList.toggle('active');
        header.setAttribute('aria-expanded', !isExpanded);
        
        if (!isExpanded) {
            body.style.maxHeight = body.scrollHeight + "px";
        } else {
            body.style.maxHeight = null;
        }
    });
});
```

#### D. Validación de Formulario y Feedback Empático
```javascript
// main.js - Validación robusta y accesible de formulario de contacto
const contactForm = document.getElementById('cepsitced-contact-form');
const feedbackContainer = document.getElementById('form-feedback');

if (contactForm) {
    contactForm.addEventListener('submit', function (e) {
        e.preventDefault();
        
        const nombre = document.getElementById('form-nombre').value.trim();
        const telefono = document.getElementById('form-telefono').value.trim();
        const especialidad = document.getElementById('form-especialidad').value;
        const mensaje = document.getElementById('form-mensaje').value.trim();
        
        // Regex simple para teléfonos de Perú (9 dígitos o 7 fijos)
        const phoneRegex = /^[0-9\s\-+]{7,15}$/;
        
        if (!nombre || !telefono || !especialidad) {
            showFeedback('Por favor, completa los campos marcados como obligatorios.', 'error');
            return;
        }
        
        if (!phoneRegex.test(telefono)) {
            showFeedback('Por favor, ingresa un número de teléfono válido para coordinar la llamada.', 'error');
            return;
        }
        
        // Simulación de envío exitoso
        showFeedback('Enviando tus datos con absoluta reserva...', 'info');
        
        // Simular llamada fetch / AJAX
        setTimeout(() => {
            showFeedback('¡Datos recibidos con éxito! Un especialista se comunicará contigo con total discreción.', 'success');
            contactForm.reset();
            
            // Disparar evento personalizado de tracking
            if (typeof trackConversionEvent === 'function') {
                trackConversionEvent('FormularioContacto_Submit');
            }
        }, 1500);
    });
}

function showFeedback(message, type) {
    if (!feedbackContainer) return;
    
    feedbackContainer.textContent = message;
    feedbackContainer.className = `form-feedback ${type}`;
    feedbackContainer.setAttribute('aria-live', 'polite'); // Accesibilidad
}
```
